ER-диаграма

Schools (1) ───< Classes (1) ───< Students
       \__________________________/
Пояснение:
- 1 → один
- < → много
- Стрелки показывают связь через Foreign Key


UML (текст)

+-------------------+
|      School       |
+-------------------+
| id                |
| school_number     |
| school_address    |
+-------------------+
         1
         │
         ▼
+-------------------+
|      Class        |
+-------------------+
| id                |
| class_name        |
| room_number       |
| school_id (FK)    |
+-------------------+
         1
         │
         ▼
+-------------------+
|      Student      |
+-------------------+
| id                |
| first_name        |
| last_name         |
| address           |
| school_id (FK)    |
| class_id (FK)     |
+-------------------+

Примеры данных (SQLite / Django)

# School
1, 123, 'м. Київ, вул. Освітня 8'

# Class
1, 1, '10-А', 203
2, 1, '10-Б', 204

# Student
1, 'Віктор', 'Орлов', 'вул. Сонячна 12', 1, 1
2, 'Марія', 'Коваль', 'м. Київ, вул. Миру 5', 1, 2
